Welcome to Ofaco

About Ofaco

The macOS Mail application offers text completion when writing mails, which can be in­voked by hitting F5. When it was introduced, it offered completions based on the list of words from the mail you were typing. Unfor­tu­nately, Apple then decided that it would be smarter to use the system dictionary for the com­ple­tion. From that day on, using the com­ple­tion in Mail meant to have to scroll through an end­less list of (partially very strange) words which are in no way related to your mail. In other words: it slowed typing down instead of making it faster.

Ofaco aims at reverting this. The name “Ofaco” stands for “Old-fashioned completion”, which says it all: it replaces Mail’s dictionary-based completion mechanism with one that uses the words from the message window from which it was invoked.

Additionally, there are default completions: many people use certain words or phrases very frequently in their mails, especially when writing job-related mails. For this reason, you can define a list of words or phrases which are always used for completing as if these words and phrases were in the mail. You can open the file in which these words and phrases are defined by trying to auto-complete “?” by hitting F5 when writing a Mail message.

Installation

Before installation, please make sure you are using macOS 10.5 (“Catalina”). This version will neither work with a newer nor with an older macOS version.

Ofaco requires a manual installation process. Earlier Ofaco versions included an installer, but due to changes in macOS, this has become more and more tedious, so I now find that maintaining the installer causes more work than maintaining Ofaco itself.
Conclusion: no installer anymore.

Installation steps:

  • Quit Mail
  • Open the Mail data folder. The path is ~/Library/Mail, and if you don’t know how to get there: hit ⌘⇧G in the Finder, then paste ~/Library/Mail into the dialog which opens, hit Returns and there it is. If this folder does not already contain a folder named “Bundles”, create it.
  • Copy Ofaco.mailbundle to the folder “Bundles”.
  • Open Terminal (if you don’t know how: hit ⌘⇧U in the Finder, and in the folder which opens, launch “Terminal”) and, and paste the following line into the window which opens:
    defaults write com.apple.mail EnableBundles -string 1
    Then, hit Return.
  • As final step, you have to enable the plugin in Mail:

    If you don’t see the “Manage Plug-Ins…” button in the preferences window, try to log out and re-login. I can’t tell why, but sometimes this is necessary.
